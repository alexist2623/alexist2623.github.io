Image Transaction System

DVI Display

Image display system with DVI/HDMI interface. Image display is controlled through double buffer scheme as shown in Image Controller module. When external camera exposure is ended, Image Controller module makes IRQ to processor and new image is loaded to Image Controller. In the same time, the back buffer discharges a new image to DVI interface, and make external camera exposure.

Left figure shows how the display is composed, and tight figure shwos interrupt routine and overall module structure. As shown in left figure, PC sends command to ZCU104 through TCP, and baremetal LWIP running on the ZCU104 parse the command, and do work. Image data is saved in SD Card and loaded to DDR4 memory. Image display system displays 100x100 MNIST image, and other part of the display is zero padded.
